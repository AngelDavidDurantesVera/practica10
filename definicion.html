Se necesita para crear

Si vamos a crear una web estática, solo con frontend, no necesitaremos ningún servidor para probarla, ya que el propio navegador la abrirá y dibujará. Pero si la web es dinámica, y tiene backend, necesitaremos un servidor, tanto en la nube como en nuestro propio PC. Para crear webs, eso sí, podemos usar tanto Windows como Linux o macOS, es indiferente.

La base de datos

La gestión y administración de las bases de datos, aunque forma parte del desarrollo backend, merece una especial mención dentro del apartado de lenguajes de programación. Dentro de la base de datos es donde se guardará toda la información. Es una de las partes críticas de una infraestructura web, ya que si se pierde o se daña, perderemos absolutamente todo. Aunque no hay lenguajes de programación como tal, hay varios tipos de bases de datos, pudiendo elegir uno u otro según las necesidades de cada web y los conocimientos de cada desarrollador.
